# Aggiornamento logo “dal 2021”

Sostituisci il file `index.html` nella root del repository GitHub con quello presente in questa cartella, poi esegui **Commit changes**.

Modifiche incluse:
- eliminata la vecchia dicitura “dal 2021” esterna, sotto il riquadro del logo;
- inserita la dicitura “dal 2021” direttamente nel logo, sopra la testa della signora, con stile calligrafico coerente con la scritta del logo;
- nessuna modifica alla posizione o alle dimensioni del riquadro logo nella webapp.
