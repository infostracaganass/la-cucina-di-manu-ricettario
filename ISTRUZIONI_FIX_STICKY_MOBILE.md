# Fix motore ricerca mobile sempre compatto

Questa versione elimina il cambio dinamico del box di ricerca durante lo scroll su cellulare.

## Modifica principale

Su schermi piccoli il motore di ricerca resta sempre compatto:

- titolo del box nascosto;
- filtri categoria nascosti;
- visibili solo campo ricerca, pulsante Cerca e pulsante Cancella;
- nessun cambiamento di forma o dimensione durante lo scroll.

## Aggiornamento

Sostituisci almeno `index.html` nella root del repository GitHub e fai commit. Vercel ridistribuirà automaticamente.
